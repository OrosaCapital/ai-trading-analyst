# Indicator Library Reference

This document summarizes the indicators implemented in `src/lib/indicators.js`, recommended uses (overlay vs subpanel), and suggested presets for common day-trading timeframes.

## Implemented Indicators

- **SMA (Simple Moving Average)**: Overlay. Presets: 20, 50, 200.
- **EMA (Exponential Moving Average)**: Overlay. Presets: 9, 20, 50.
- **RSI (Relative Strength Index)**: Subpanel. Presets: 14.
- **MACD**: Subpanel. Presets: fast=12, slow=26, signal=9.
- **VWAP**: Overlay (session-based). Presets: session reset daily intraday.
- **ATR (Average True Range)**: Subpanel. Presets: 14.
- **Bollinger Bands**: Overlay. Presets: period=20, stdDev=2.
- **Stochastic (%K/%D)**: Subpanel. Presets: k=14, d=3.
- **CCI (Commodity Channel Index)**: Subpanel. Presets: 20.
- **ROC (Rate of Change)**: Subpanel. Presets: 12.
- **OBV (On-Balance Volume)**: Subpanel. Presets: n/a (cumulative)
- **CMF (Chaikin Money Flow)**: Subpanel. Presets: 21.
- **ADX (Average Directional Index)**: Subpanel. Presets: 14.
- **Parabolic SAR**: Overlay. Presets: step=0.02, maxStep=0.2.
- **TRIX**: Subpanel. Presets: period=15, signal=9.
- **Keltner Channels**: Overlay. Presets: ema=20, atr=10, mult=1.5.

## Recommended Timeframe Presets

Use these as starting points for intraday trading. Adjust sensitivity by shortening/lengthening periods.

- 1m: faster settings — EMA 9/20, RSI14, Stochastic (14,3), MACD (12,26,9)
- 5m: EMA 20/50, SMA 50, RSI14, Bollinger (20,2)
- 15m: EMA 20/50, SMA 200, ADX14, Keltner (20,10,1.5)
- 1h: SMA50/200, EMA20, VWAP for session context, MACD for trend

## Overlay vs Subpanel Guidance

- Overlay: moving averages, VWAP, Bollinger, Parabolic SAR, Keltner
- Subpanel: RSI, MACD, ATR, ADX, OBV, CMF, CCI, Stochastic, TRIX, ROC

## Notes

- Implementations are optimized for clarity and didactic use — production systems should validate edge cases and ensure numerically stable smoothing for very large datasets.
- All functions expect arrays of numbers or arrays of candle objects (for volume/high/low/close).
